#### new in c:

in c program:
*=printf=* function: left/right-justify printf output 

```c
// to right
printf("%<width><type>")
printf("%3lld",a);
```

```c
// to left
printf("%-<width><type>")
printf("%-3lld",a);
```


| 类型      | 类型代号 |
| --------- | -------- |
| int       | d        |
| long long | lld      |
| char      | c        |
